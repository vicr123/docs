Importing CDs

theBeat allows you to import CDs into your computer, so that you are able to listen to them without needing the CD in the drive.

Note

Importing tracks from a CD is only supported on Linux platforms at this time.

Play Nice!

Tracks that you import from a CD are still protected by copyright law.

To start importing a CD, navigate to the CD and select Import.

Import Options#

  • Folder
    The folder to place files imported from the CD
  • Format
    The format to import files as

Once these options are set as you'd like, select Import to start importing the tracks. The import progress can be tracked using the Jobs button in the top right corner of the window. Once the import process is complete, a notification will appear to let you know.
